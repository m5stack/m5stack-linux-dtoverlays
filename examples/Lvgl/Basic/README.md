# CM4Stack_LvglDemo

Lvgl Demo for CM4STACK

![](https://github.com/m5stack/m5stack-linux-dtoverlays/blob/main/examples/Lvgl/Basic/pic.png?raw=true)

#### Update submodules

```
git submodule init
git submodule update
```

#### Build

```
mkdir build && cd build
cmake .. && make
```

#### Run

```
./cm4LvglDemo
```
